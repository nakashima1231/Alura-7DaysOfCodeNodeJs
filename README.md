# Alura 7 Days Of Code

## Node.js, Express & APIs

Este repositório reúne projetos desenvolvidos em **Node.js**, com foco na criação de **APIs REST** utilizando **Express** e integração com banco de dados, como parte da iniciativa de aprendizado da plataforma Alura.

O objetivo é aplicar conceitos fundamentais de desenvolvimento backend na prática, explorando organização de projetos, criação de rotas, controladores, modelos, conexão com banco de dados e boas práticas de arquitetura para aplicações web.
